## Emergency Procedures

### Service Recovery
- CopyParty restart: pkill copyparty && /start-copyparty-enhanced.sh
- API restart: Restart Audio Portrait API service
- DNS issues: Check /etc/hosts and DNS settings
- Port conflicts: Check ss -tlnp for port usage

### User Issues
- Password reset: Update in CopyParty configuration
- Access problems: Check directory permissions
- Upload failures: Verify disk space and permissions
- Connection issues: Verify network and firewall settings
